1. Disable Startup Programs

  1. Press Ctrl + Shift + Esc
  2. Open Startup tab
  3. Disable unwanted apps
Reduces load when PC starts → faster boot.

2. Delete Temporary Files

  1. Press Win + R
  2. Type "%temp%"
  3. Delete all files
Removes junk → improves speed.

3. Best Performance Mode

  1. Search sleep
  2. Select peformance mode
cpu get more power -> fast.

4. Increase Virtual Memory

  1. Open Advanced system settings
  2. Go to Performance settings
  3. Set custom size
Helps when RAM is low.

5. Use SSD

  1. Buy SSD
  2. Install OS in SSD
  3. Use HDD for storage
Biggest speed improvement.
